# Design System — Urukais Klick

## Paleta de colores

- **Primary:** Deep Purple `#673AB7`
- **Secondary:** Teal `#009688`
- **Background:** `#FAFAFA`
- **Surface:** `#FFFFFF`
- **Error:** `#B00020`

## Tipografía

- **Headlines:** Inter / Roboto (Flutter: GoogleFonts)
- **Body:** Inter / Roboto

## Componentes base

- `UkButton` — Primario, secundario, outline, ghost
- `UkInput` — Texto, email, contraseña, búsqueda
- `UkCard` — Producto, contenido, perfil
- `UkAvatar` — Imagen de usuario con fallback
